Double Check — all-formats fixture

Focus any field and press the shortcut (Shift+Command+Space on a Mac — shown by Chrome as ⇧⌘Space — or Ctrl+Shift+Space elsewhere). Pre-filled fields open in verify mode; empty ones in input mode. Each field lists a valid sample to type and an invalid one to see the red diagnosis. For screen-region scanning, serve this page over http (python3 -m http.server in the repo root, then localhost:8000/tests/pages/all-formats.html) — or enable “Allow access to file URLs” for the extension if you open it as a file. To exercise Submit Guard: add this page’s origin in Settings, verify a field, then try Send.
